# Proyecto DISTRITO_CERO

## Descripción General
DISTRITO_CERO es un proyecto que organiza y estructura recursos estáticos y plantillas HTML para un sitio web. Este proyecto incluye hojas de estilo CSS, imágenes, íconos, scripts de JavaScript y plantillas HTML para diferentes secciones del sitio.

## Estructura del Proyecto
La estructura del proyecto es la siguiente:

```
README.md
static/
	css/
		stilocole.css
		stilocon.css
		stiloser.css
		styles.css
	icon/
	img/
	JavaScript/
templates/
	coleccion.html
	contacto.html
	index.html
	servicios.html
```

### Descripción de Carpetas y Archivos

- **README.md**: Archivo de documentación del proyecto.
- **static/**: Contiene los recursos estáticos del proyecto.
  - **css/**: Carpeta con las hojas de estilo CSS.
    - `stilocole.css`: Estilos específicos para la página de colección.
    - `stilocon.css`: Estilos específicos para la página de contacto.
    - `stiloser.css`: Estilos específicos para la página de servicios.
    - `styles.css`: Hoja de estilos general para el sitio.
  - **icon/**: Carpeta para íconos utilizados en el sitio.
  - **img/**: Carpeta para imágenes utilizadas en el sitio.
  - **JavaScript/**: Carpeta para scripts de JavaScript.
- **templates/**: Contiene las plantillas HTML del proyecto.
  - `coleccion.html`: Plantilla para la página de colección.
  - `contacto.html`: Plantilla para la página de contacto.
  - `index.html`: Plantilla para la página principal.
  - `servicios.html`: Plantilla para la página de servicios.

## Cómo Usar el Proyecto
1. Clona este repositorio en tu máquina local.
2. Abre las plantillas HTML en un navegador para visualizar las páginas.
3. Modifica los archivos CSS o HTML según sea necesario para personalizar el sitio.
